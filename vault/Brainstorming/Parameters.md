# Parameters (brainstorming lens)

When evaluating any part of Drishya, run it through the axes:
- **Purpose / Structure / Surface** — see [[Purpose Structure Surface]].
- **Level** (0–5) — see [[Border grammar levels]].
- **Capability preserved?** — does the change keep every existing function?
- **Trace** — does it respect "who saw what / who wrote what" (`origin`)?

Open creative threads to develop later live in [[Slash command potentialities]] (ghost text, versions, region-aware writing, trace view). Park new ideas there so they're not lost and can be built without breaking [[Constraints we must not break]].
