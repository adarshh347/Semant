# Drishya and Semant

**Semant** is the project. **Drishya** is the page where one image is dwelt upon — read, marked, and written into a story. It replaces the old `PostDetailPage` (rename to `DrishyaPage` scheduled for a build pass).

Vocabulary:
- **Drishtikone** — the app / "the way of seeing" (being rebranded; `-kone` sounds dated → frontrunners **Drishti** / **Nazar**).
- **Sutradhar** — the thread-holder; the writing/AI companion identity.
- **Aletheia** — the machine reading of an image (lenses, concealed, uncertainty).
- **Darpan / personas** — source accounts an image can feed.

Layout: a split workspace — **Visual pane** (image + annotations) | **Content pane** (Story / Highlights / Unconceal). See [[Purpose Structure Surface]] for the page's purpose statement.

The six architectural lanes: 1 top chrome · 2 split shell · 3 visual+annotation · 4 content view · 5 writing studio *(in progress)* · 6 unconceal. Tracked as GitHub issues; see [[How we work]].
