## spark-09 (NEW) — relational vocabulary is an attractor, and the noun is the variable

**Claim, at the narrowest scope that survives.** On one pair of images, asking what **motif** each
carries elicited an assertion of shared motif; asking how each is **put together** elicited shared
properties **plus an explicit denial of sameness**; asking neither elicited no cross-image claim at
all. Everything else was held constant, including co-presence and the symmetric share/only-one-of-
them demand. **The kinship was in the question.**

**Status: SPARK at n=1 pair, 1 model, 3 calls.** Sibling to spark-06, which resolved the same shape
for *address* vocabulary. Two vocabularies now show the effect; that is a pattern worth naming and
not yet a disposition.

**What would strengthen it:** the fourth arm (§9.3) — *"say what motif each image carries"* with no
relational clause — which separates the **noun** from the **identity presupposition**. One call.
Then a second pair. Then other relational vocabularies (*influence*, *lineage*, *belongs to the same
tradition*), each of which is currently open.

**What would kill it:** arm A's hedge reproducing under arm C's wording on a different pair, which
would make it a register effect (abstraction, not kinship) rather than a vocabulary effect. See
`critique.md` §6.

**Relation to product:** none authorized. This is a fact about prompting a VLM, not a fact about
Semant's ontology.

## spark-08 — gains a PROSPECTIVE, controlled corroboration

The cycle-6 audit of runs 001–006 found identity-reaching claims clustered by **question type**:
they appeared in exactly one run of six, the only one asked an open *"what is happening in this
photograph?"*.

**This run reproduced that prospectively, within a single controlled comparison.** Same two images,
same call shape, three questions. The arm asked *"say what it is"* produced **3 identity-reaching
rows including both monument names** (Shah Mosque, Sainte-Chapelle). The motif arm produced 2 and
named no monument. The construction arm produced 1.

That is stronger evidence than the audit's, because the audit compared across runs that differed in
fixture, budget and date, while this compares three questions against **identical bytes**.

**Still a SPARK.** n is one run; the arms differ in more than question-type; and this was a
side-observation of a run designed for something else. **It is corroboration, not a second
experiment.**

**Consequence for the convention:** the cycle-7 amendment keying HW-C5 §6.1's overturn test on the
`kind` column earned its keep on its first run — this ledger's 13 rows split 6 identity-reaching /
7 low-grade, a distinction the pre-amendment "empty vs non-empty" test could not have seen.

## Unresolved questions

1. **Is it the noun or the presupposition?** Arm A moves *motif* and *sharing* together. The fourth
   arm separates them and is one call. **This is now the highest-value single call available to the
   program**, because cell 1 makes the noun load-bearing.
2. **Is arm A's hedge a kinship effect or an abstraction effect?** *"Creating a sense of the divine
   or celestial"* may be what this model produces whenever asked an abstract question about
   ornament, with sameness the only place the abstraction can land.
3. **Would a counterbalanced order (A → C → B) reproduce the asymmetry?** All calls are stateless,
   so there is no carry-over mechanism — but the program has been burned once inferring absence of a
   bug from code shape.
4. **Does A6's stage-2 device — a proposition attributed to a third party — behave like a question
   frame at all?** Untested by this run, and untested by A5, whose sycophancy control was never
   exercised because the model never dissented.
5. **Do other relational vocabularies pull?** *Influence*, *lineage*, *derived from*, *belongs to
   the same tradition*. Open, exactly as this run's own question was left open by 007.
6. **Is `KIN-ASSERTED` or `KIN-HEDGED` the right score for arm A?** Immaterial to the grid cell,
   material to how badly A6 is exposed. See `critique.md` §1.
