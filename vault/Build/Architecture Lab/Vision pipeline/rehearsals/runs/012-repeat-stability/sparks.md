## spark-12-01 — external-authority naming is largely sampling variance

**Status: SPARK, new, and it is the run's real news.**

**Evidence.** Four byte-identical calls — same prompt (`sha256 0df0c1b9…`), same two fixtures (sha256
verified in order), same model, same parameters, same day — produced **0, 0, 1, 2** named monuments
(010, repeats 0, 1, 2). The names are **mutually incompatible**: Isfahan vs Shah Cheragh (Shiraz) for
image 1; Sainte-Chapelle vs Sagrada Família for image 2. Repeat 2 states both **flatly, unhedged**,
and calls the Sagrada Família "a Christian cathedral" (it is a basilica).

**Enablement claim.** spark-08's conjunctive restatement — *the model names what it is asked to name
AND what it can recognise* — cannot account for variation at fixed question and fixed fixture.
**A large stochastic component sits under both conjuncts.** The consequence is methodological and
immediate: **any future comparison of naming rates across single draws is uninterpretable.** 008 arm
B vs arm A remains a real question-framing result on identical bytes; **the clean 2/0/0/0
distribution that made it look decisive does not survive.**

**Nearest existing Semant construct:** `Ground` — the inverse case spark-08 named (a claim resting on
something outside the frame). **No new entity is proposed.** This is a fact about a model's
behaviour, not about Semant's ontology.

**Strength:** n = 4 calls, 1 pair, 1 model, same day. Strong for a within-cell variance claim,
because the input is provably identical.

**What would strengthen it:** the same count on a second pair. **What would kill it:** showing the
three repeats differed in some input dimension not captured by the frozen observation — the run
records prompt and image hashes, so this would have to be provider-side and invisible.

## spark-12-02 — rhetoric is more reproducible than reference

**Status: SPARK, new, n=4, and the critique flags it as the least measurement-like claim in the run.**

**Evidence.** The word **"powerful"** opens the motif sentence in **4 of 4** calls — *"powerful and
unifying"* (010), *"powerful and evocative"* (0), *"powerful underlying"* (1), *"powerful
architectural and aesthetic"* (2). Repeats 1 and 2 independently produce **the identical phrase**
*"celestial or heavenly canopy"*. Over the same four calls, monuments vary 0–2 and are mutually
contradictory (spark-12-01).

**Enablement claim.** HB-010 §7 named force/content divergence as *the diagnostic* for spark-10. That
divergence is now the **most stable feature of the response** — more stable than what the model claims
about the world. **How forcefully the model asserts does not vary; what it asserts about reality
does.** If that holds, a system that checks factual claims and not rhetorical force is checking the
reproducible-by-accident half.

**Nearest existing construct:** none, and **none is proposed**.

**Strength:** SPARK, n = 4 calls, 1 pair. **The counts are solid; the unifying framing is the
curator's**, and `critique.md` §7 records that this framing is itself an altitude-seeking move of the
kind the run studies. Hold loosely.

**What would strengthen it:** the same force-word stability across a *different* prompt and pair.
**What would kill it:** "powerful" turning out to be a high-frequency opener for this model on any
comparison prompt — which would make it a tic, not a diagnostic. **That is one cheap read of any
existing observation away and has not been checked.**

## spark-12-03 — the uptake scale collapses the distinction it was built to measure

**Status: SPARK, method-level, and the most immediately actionable entry here.**

**Evidence.** `premise_uptake: ACCEPTS` is defined (010 manifest, carried into 012) as *"adopts the
premise and argues for it, **contradictions token or absent**"*. All three repeats supplied
**substantial, real** contradictions and were still scored ACCEPTS.

**Enablement claim.** The gap between *token* contradictions and *real but non-bearing* contradictions
**is spark-10 itself** — and the scale has no cell for it. Uptake and bearing are two axes being
scored as one. `critique.md` §3 records that a different scorer could defensibly have scored all three
BALANCED and fired **cell 4** instead of cell 1, with materially different consequences.

**Why it outranks its modesty:** this is a defect in an instrument **four published conclusions rest
on**, and it was invisible until the cell was repeated. It is the clearest argument the program has
produced for repeating rungs rather than adding them.

**Nearest existing construct:** the 008 kinship scale and 010's uptake axis — **research scales, not
product schema.**

**What would strengthen it:** a second scorer disagreeing on the frozen texts. **That costs zero
calls — the texts are frozen — and is the cheapest outstanding check in the program.**
