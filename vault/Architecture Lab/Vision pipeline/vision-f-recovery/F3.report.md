## Approach

Per-Region honest recovery (`backend/services/geometry_recovery.py`), driven through the F1
ledger + backup with the F2 skip-guard:

- `retain_mask` — already authoritative; untouched.
- `derive_polygon` — a GENUINE traced polygon (≥ 8 pts) rasterized to a canonical mask.
- `sam_refine_box` — box-only evidence: SAM segments the real pixels from the box **prompt**
  (genuine, never fabricated), keeping the Region id + label + curator fields.
- `retain_box` — honest fallback when SAM is down.

geometry under `_prev_geometry`, and preserves every curator field. Detached grounds are flagged
`detached=True` visibly, never deleted.
