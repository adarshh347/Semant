# Architecture Lab (moved from `architecture-lab/`)

The project's planning brain, reorganised into the vault. `architecture-lab/` at the repo root is retired — everything lives here now.

## Structure
- **Findings/** — research, analysis, track findings, OSS studies (`responses/…`, `track-*`, `*.findings.md`).
- **Build specs/** — executable prompts for Claude Code (`*.build.md`, `*.prompt.md`, lanes, `issues/`).
- **Decisions/** — `decisions-log`, `workflow-log`, `workflow-protocol`, architecture map, purpose lens.
- **Vision pipeline/** — the Darshan initiative: `vision-initiative/`, the `vision-*` agent-run reports, rehearsals.
- **Plans/** — loose plans (features, field/differential, finetune, master plan).
- **Landing articles/** & **Orchestrator guide/**.

## Convention (also in root `CLAUDE.md`)
New planning/architecture/findings/build/decision/vision docs are authored **here in the vault**, not in a recreated `architecture-lab/`. Commit as `docs(vault): …`.
