# R0 — Breadth portfolio (six rehearsals, proposed for R2)

Per `04-rehearsal-repertoire.md §Batch A` and the R0 direction: six rehearsals spanning **different
families**, visual domains, source roles, single- and cross-image, ≥1 non-figurative, ≥1 sensory
disagreement, ≥1 narrative overreach, and **≥2 expected stalls/refusals**. Goal: discover candidate
*kinds*, not confirm Passage 001. **Proposed only — not run in R0.** Fixtures prefer connected posts
(F-recovered evidence); where a domain is missing, fixture data is authored with explicit provenance
(never fake production records).

## Coverage check against the R0 requirements

- **Families:** R4, R5, R7, R8, R9, R12 — six distinct families (not six gesture variants). ✓
- **Domains:** sculpture, architecture, fashion, abstract/textile, painting. ✓
- **Source roles:** spatial lens, counterforce, discriminator, practice lens, ethical limit,
  misleading source — deliberately varied; **A3 explicitly swaps away from Merleau-Ponty** to test
  source ventriloquism. ✓
- **Single vs cross-image:** A1/A2/A4/A5 single; A3/A6 cross-image. ✓
- **Non-figurative:** A4. ✓ · **Sensory disagreement:** A2. ✓ · **Narrative overreach:** A5. ✓
- **Expected stalls/refusals (≥2):** A1 (interval may lack evidence), A5 (may refuse the sentence),
  A6 (expected refusal). ✓ (three)
- **Sequence/quality controls:** A3 and A6 each get a **sequence-inversion replay** and a
  **run-without-source** pass (foundry required tests) to check the discovery is in the material,
  not the ordering or the philosophy.
- **Small enough for close human review:** ≤ 2–3 images each, connected fixtures where possible. ✓

## What the portfolio must NOT do

Not validate Passage 001's constructs (A3 is a *transfer/negative* test, not a confirmation); not
graduate any candidate in its originating run; not write to production; not create Atlas/Codex; not
introduce a schema to make a trace look elegant.
