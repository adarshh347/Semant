# Semant — working conventions

## Docs live in the vault (not architecture-lab)

`architecture-lab/` is **frozen and has moved** into the Obsidian vault at `vault/Architecture Lab/`.
From now on, author ALL planning, findings, build-spec, decision, and vision docs in the vault:

- Findings / research / analysis  → `vault/Architecture Lab/Findings/`
- Build specs / execution prompts  → `vault/Architecture Lab/Build specs/`
- Decisions / workflow / protocol  → `vault/Architecture Lab/Decisions/`
- Vision pipeline / agent reports   → `vault/Architecture Lab/Vision pipeline/`
- Loose plans                       → `vault/Architecture Lab/Plans/`
- Design language / taste           → `vault/Frontend Analysis/` and `vault/Design/`

Do NOT recreate `architecture-lab/`. Commit docs as `docs(vault): …`.
